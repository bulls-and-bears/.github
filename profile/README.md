## 프로젝트 소개 📝

`Bulls & Bears`는 주식 초보자들이 쉽고 효과적으로 배당금 수익을 예측할 수 있도록 도와주는 서비스입니다.
주요 기능은 다음과 같습니다:

- 사용자가 보유하거나 관심 있는 주식에 대한 배당금 수익을 예측
- 주식 초보자를 위한 인터랙티브한 투자 시뮬레이션 제공
- 실시간 데이터와 과거 데이터를 활용한 상세한 분석 보고서 생성
- 다양한 시각화를 통한 직관적인 데이터 이해

`Bulls & Bears`는 주식 투자에 대한 이해도를 높이고, 개인화된 재무 목표 설정에 기여하는 것을 목표로 합니다.
